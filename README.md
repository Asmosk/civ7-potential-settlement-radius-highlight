# Civ7 Potential Settlement Radius Highlight

![Logo](https://github.com/Asmosk/civ7-potential-settlement-radius-highlight/blob/main/images/nyaka-potential-settlement-radius.png?raw=true)

A QOL UI mod for Civilization VII. When settler lens is active, hovering over a tile will highlight all tiles around it that would be workable from settlement in that tile.
Easy to see what resources you would be able to reach from it.
Settler lens is activated automatically when you select a settler unit, or you can enable it manually from the lens menu.

Should work with any other mod without conflicts.
